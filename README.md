# pregunta 1 .gitignore

gitignore sirve para ignorar un archivo en el repo con el fin de mantener el
repo limpio y/o proteger informacion sensible

# como utilizarlo

- crear un archivo llamado .gitignore
- en ese archivo escribir el prefijo de los archivos que quieres ignorar
- puedes escribir de manera especifica, el nombre completo del archivo para ignorarlo

# tema 3 codigo misterioso

el codigo constaba de 3 funciones que modificaban un numero inicial
- la primera funcion invertia el numero inicial, en este caso paso de 452 a 254, cambie el nombre de la funcion a **invertirNumero** y sus variables a **numeroTemporal** y **nRevertido**
- la segunda funcion agarro el numero invertido y lo dividio en 2, paso a llamarse **nDividido**
- la tercera funcion guardaba el numero en una variable temporal y esta iba sumando todos sus componentes, al final sumaba el resultado con el numero dividio. la funcion llame **suma de instancias**
